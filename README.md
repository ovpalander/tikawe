# Talonaattori - taloyhtiön hallintasovellus

## Sovelluksen ominaisuudet

* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään.
* Käyttäjä pystyy määrittelemään taloyhtiön ominaisuudet
    * Y-tunnus
    * Rakennusvuosi
    * Osakkaat
    * Huoneistojen lukumäärän määrittely.
    * Huoneistojen ominaisuuksien määrittely (pinta-alat, osakenumerot, osaakkeenomistajat)
* Taloyhtiön tietojen katselu.
* Osakkaiden käyttäjätunnusten/oikeuksien hallinta (yhtiön luonut osakas on ylläpitäjä, voi antaa ylläpitäjän oikeudet muille).
